① ターン　② ゴール（/goal）

1

ターンベース

どこに何を21
ClaudeCodeチャンネルClaude Code を学ぶ
第4章

迷ったときの判断の軸 ✅

  • 「毎回必ず」「絶対にするな」→ 指示ではなく Hook（PreToolUse）＋権限で止める
  • 指示（CLAUDE.md/Rules）は長セッションやインジェクションで破られうる
  • 30行を超える手順 → 事実置き場ではなく Skill へ
  • 中間結果を見たくない重い調査 → Sub Agent に隔離（結果だけ戻る）
  • Skill と Sub Agent は「手順を見せたい／隔離して結果だけ受け取りたい」で選ぶ

Loop入門の要点

  • ループ＝停止条件を満たすまでサイクルを繰り返す。まずシンプルに始める
  • 4種類（ターン／ゴール/goal／タイム/loop・/schedule／プロアクティブ）を使い分ける

  • チームなら北極星と名簿、信頼は段階的に広げる
  • 実践は「見極め→最初の1個→失敗/コスト管理」。今日1つ回すのが第一歩
